# IRONPEAK Gym Website 🔥

Welcome to the **IRONPEAK Gym** website, an ultra-premium, high-contrast, fully interactive landing page designed for professional-grade fitness centers and athletic training clubs. 

This is a modern front-end portfolio piece built entirely with semantic **HTML5**, custom structured **Vanilla CSS3**, and highly optimized **ES6+ JavaScript** (zero external framework dependencies!).

---

## ⚡ Features & Interactive Highlights

- **Sleek Dark Mode & Glassmorphism:** Features a premium obsidian and carbon-black color palette with vibrant neon-orange highlights and frosted-glass card overlays.
- **Custom Cursor & Smooth Trailing Ring:** Uses an interactive custom cursor system with a fast-following dot and a smooth trailing outer ring that morphs, scales, and shifts color dynamically when hovering over clickable items.
- **Parallax Atmospheric Hero:** Large responsive cover layouts displaying high-intensity generated athletic photography, featuring a rotating key-branding typography reel.
- **Micro-Animations & Hover Effects:** Smooth, responsive scaling transitions and neon gradient border glow states on training and pricing cards.
- **Interactive Pricing Switcher:** A styled sliding toggle switch that dynamically recalculates Monthly vs. Annual membership pricing with smooth mathematical integer counters.
- **Touch & Mouse-Swipeable Testimonials:** A custom-engineered testimonials carousel supporting mouse-drag and mobile swipe gestures with dynamic spring-back boundary physics and automated dot pagination.
- **Viewport Scroll Reveals:** Employs high-performance `IntersectionObserver` loops to elegantly fade and slide sections into view as you navigate the page.

---

## 🛠️ Built With

- **Structure:** Semantic HTML5
- **Style:** Vanilla CSS3 (Custom Variables, Flexbox, CSS Grid, Media Queries)
- **Logic:** Vanilla ES6+ JS (Intersection Observer API, RequestAnimationFrame LERP animations, custom drag handlers)
- **Visuals:** Custom vector SVG iconography and AI-generated professional athlete imagery.

## 🚀 Quick Local Setup

No build tools or installations are required! You can open the project directly in your browser.

1. Clone or download the repository files.
2. Double-click the `index.html` file, or open it from your terminal:
   ```bash
   open index.html
   ```

## 🌐 How to Deploy

### 1. GitHub Pages (Free)
1. Push this folder to a new **Public** GitHub repository.
2. Go to **Settings ➔ Pages** tab.
3. Set your Source to **Deploy from a branch**, select the **`main`** branch (`/root` folder), and click **Save**.
4. Your site will be live at `https://<YOUR-USERNAME>.github.io/ironpeak-gym/`.

### 2. Vercel
1. Log in to Vercel and go to the dashboard.
2. Drag and drop your **`Ironpeak gym`** project folder directly into the Vercel Deploy upload box, or connect your GitHub repository for automated continuous deployments!
